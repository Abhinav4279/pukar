# pukar
## Description
A stand alone crowdfunding java applet with admin and user access. Application that stores the information about donors/donations and provide access to this information in tabular format to the administrator. Database requirements are handled using MySql RDBMS and connectivity is implemented using JDBC(Java Database Connectivity) driver.
